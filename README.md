A home status screen built with a raspberry pi and pi touch display 2. I wrote it in c++ and qml. 

To use, add an open-meteo API key for weather, and your calendar's public link to config.json, which should sit next to the binary. There should also be a pictures/ folder next to the binary as well to house pictures that are cycled through by the photo frame in the interface.

I left an example config.json that you just need to add the URLs into.

Photos should be .png .jpg or .jpeg.
